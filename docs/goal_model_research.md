# Independent international goal model research

Status: work in progress.

This branch is isolated from the mobile dashboard rearchitecture. It does not modify dashboard files, deployment configuration, services, cron, Caddy, firewall rules, or Hermes configuration.

The research pipeline will progress through the requested phases: data audit, transparent Poisson baselines, recency weighting, low-score correction, shrinkage/hierarchical approximations, controlled squad priors, measurable tournament-state effects, chronological backtests, and an evidence-based integration recommendation.
